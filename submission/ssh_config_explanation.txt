| Parameter                  | Nilai Saat Ini / Status                                                         | Keterangan                                                                                                                                                 |
| -------------------------- | ------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Port**                   | `#Port 22` (dikomentari, default 22)                                            | Port TCP yang digunakan oleh SSH. Jika dikomentari, default adalah **22**.                                                                                 |
| **PermitRootLogin**        | `#PermitRootLogin prohibit-password` (dikomentari, default `prohibit-password`) | Mengatur apakah root bisa login lewat SSH. `prohibit-password` berarti root **tidak bisa login menggunakan password**, tapi bisa menggunakan kunci publik. |
| **PasswordAuthentication** | `#PasswordAuthentication yes` (dikomentari, default yes)                        | Menentukan apakah login menggunakan password diperbolehkan. Saat ini masih default **ya**, tapi belum diaktifkan secara eksplisit.                         |
| **PubkeyAuthentication**   | `#PubkeyAuthentication yes` (dikomentari, default yes)                          | Menentukan apakah autentikasi menggunakan kunci publik diperbolehkan. Default **ya**.                                                                      |
| **MaxAuthTries**           | `#MaxAuthTries 6` (dikomentari, default 6)                                      | Batas jumlah percobaan autentikasi sebelum server menutup koneksi. Default adalah **6 kali**.                                                              |
